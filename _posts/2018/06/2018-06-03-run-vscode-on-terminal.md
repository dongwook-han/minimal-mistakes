터미널에서 특정 폴더로 이동 후, VSCode를 실행하고자 할 때 `code [Path]`로 실행합니다.

만약 `code`가 없다면 `⇧⌘P`를 눌러 Command Palette에서 `Shell Command: Install 'code' command in PATH` 를 통해 `code` 를 설치할 수 있습니다.
